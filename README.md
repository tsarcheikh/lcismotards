# LCIS Motards — Site Officiel 🏍️🏁

Bienvenue sur le dépôt officiel du site **LCIS Motards**, le club de sensibilisation routière de référence au Sénégal. Ce projet est une One-Page moderne, immersive et entièrement responsive, conçue pour promouvoir la sécurité et la discipline sur nos routes.

## ✨ Fonctionnalités Clés

- **Design Premium Light Theme** : Une esthétique propre, professionnelle et aérée, mettant en avant l'identité visuelle du club.
- **Moteur d'Animation Moto** : Une animation fluide et bidirectionnelle d'une moto au bas de la page, synchronisée avec le défilement de l'utilisateur.
- **Full Responsiveness** : Entièrement optimisé pour mobiles (iPhone SE, Android), tablettes (iPad) et ordinateurs de bureau.
- **Intégration WhatsApp** : Un formulaire de recrutement intelligent qui génère un message personnalisé et redirige directement vers le contact officiel du club.
- **Expérience Utilisateur Polie** : Navigation fluide, transitions douces et icônes SVG haute qualité.

## 🛠️ Stack Technique

- **Frontend** : HTML5 moderne, CSS3 (Custom Glassmorphism), JavaScript (Vanilla ES6).
- **Styling** : [Tailwind CSS CDN](https://tailwindcss.com/) pour une gestion efficace des utilitaires UI.
- **Animations** : [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) pour les effets de révélation au défilement.
- **Favicon & Identité** : Logo officiel configuré pour une reconnaissance immédiate.

## 🚀 Installation & Déploiement

Le site est conçu pour être hébergé facilement sur **GitHub Pages**, **Vercel** ou **Netlify** sans aucune configuration serveur.

1. Clonez le dépôt : `git clone https://github.com/tsarcheikh/lcismotards.git`
2. Ouvrez `index.html` dans votre navigateur.
